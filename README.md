## 🧪 Projects

### 🏠 Home SOC Lab
A self-built Security Operations Center lab for hands-on detection and monitoring practice.
- **Stack:** VMware Workstation, Windows 11 VM, Kali Linux VM, Ubuntu Server 24.04
- **SIEM:** Wazuh (agent-based log collection, dashboard-driven monitoring)
- **Highlights:** deployed a full Wazuh SIEM, connected a monitored Windows endpoint, troubleshot networking/resourcing issues end-to-end, and validated detection using Kali-based scans (Nmap, Wireshark, tcpdump, Netcat)
- **Status:** ✅ Complete
